# 前端算法学习

使用 TypeScript 完成算法内容。
借此可以巩固 TypeScript 一些基本使用方法。

vitest 作为测试工具。
便于查看代码运行是否正确，便于代码重构。

## 运行&测试

```
// 运行所有测试
pnpm test-all
```

```
// 运行单个测试
pnpm test xxx
xxx 为单元测试名称
```
